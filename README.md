# egovEnterprise
전자정부 프레임워크 egovFrameWork Template Project(Enterprise Business) 초기 프로젝트 세팅 - 내부업무 기능 구현시 필수적인 부분만 사용 가능하도록 경량화 된 실행환경 제공
* 제공기능 : 메인 페이지, 업무사용자 관리, 공지사항 관리, 게시판 관리, 권한 관리, 프로그램 관리, 메뉴 관리
